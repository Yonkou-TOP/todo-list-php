#Aplikasi To-Do List 

## Deskripsi 
Aplikasi sederhana berbasis PHP untuk mencatat tugas harian 

## Fitur
-Tambah Tugas
-Tandai tugas selesai
- Hapus Tugas

## Struktur Folder 
- index.php

## Cara Menjalankan 
1. salin folder ke 'htdocs/'
2. jalankan XAMPP dan buka di 'https://localhost/todolist'
